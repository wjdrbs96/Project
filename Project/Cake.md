## `사용된 도구`

- [Java]() - Java 15
- [Spring Boot]() - Spring Boot 웹 프레임워크
- [Maven]() - 의존성 관리 프로그램
- [Tomcat]() - 웹 애플리케이션 서버
- [MyBatis]() - SQL Mapper
- [MySQL]() - DataBase
- [AWS EC2]() - 클라우드 환경 컴퓨팅 시스템
- [AWS RDS]() - 클라우드 환경 데이터베이스 관리 시스템
- [AWS S3]() - 클라우드 환경 객체 스토리지
- [JWT]() - 토큰 기반 인증, 인가
- [Git]() - 버전 관리 시스템
- [Tool]() - Github, Notion, Slack
